# Cybersecurity-Home-Lab
This repository documents the planning, build, configuration, testing, and development of my personal cybersecurity home lab.
The lab is designed to support learning in:

- Networking
- Virtualisation
- Windows and Linux administration
- pfSense firewalling
- Packet analysis
- SIEM/logging
- Vulnerability scanning
- Ethical hacking and CTF practice
